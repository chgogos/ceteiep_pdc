```
	$ mpicc mpi_example01.c -o mpi_example01
```

```
	$ mpirun -n 5 ./mpi_example01

	Process 1 of 5 says hello.
	Process 2 of 5 says hello.
	Process 3 of 5 says hello.
	Process 4 of 5 says hello.
```
